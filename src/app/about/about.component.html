About Project

This project is a Twitter sentiment analysis tool built with Angular. It uses the Twitter API to retrieve tweets related to a particular topic, and then performs sentiment analysis on those tweets to determine whether they are generally positive, negative, or neutral.

FEATURES

Our Features & Services.

SEARCH

Search for tweets related to a particular topic.

ANALYSIS

View sentiment analysis results for the searched tweets Filter the results by sentiment type (positive, negative, or neutral)

VISUALISATION

Visualize sentiment analysis results using a pie chart,bar chart.
